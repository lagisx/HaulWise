# Биржа грузоперевозок

Десктопное приложение для размещения и поиска заявок на грузоперевозку.
Грузовладельцы публикуют грузы, перевозчики находят подходящие заказы.

## Стек технологий

- **Java 17 + JavaFX** — клиентский интерфейс
- **Supabase (PostgreSQL)** — облачная база данных, REST API
- **Supabase Realtime** — чат в реальном времени (WebSocket)
- **Supabase Storage** — хранение изображений грузов
- **Resend API** — отправка OTP-кода на email
- **Яндекс Карты** — просмотр маршрута груза
- **OkHttp + Gson** — HTTP-запросы и работа с JSON
- **Apache Maven** — сборка проекта

## Функциональность

### Гость
- Просмотр списка грузов
- Фильтрация по маршруту, весу и цене

### Пользователь (Логист / Грузовладелец)
- Регистрация и вход в аккаунт
- Публикация, редактирование и удаление своих грузов
- Добавление грузов в избранное
- Чат с другими пользователями в реальном времени
- Просмотр маршрута на карте
- Восстановление пароля через email
- Редактирование профиля

### Администратор
- Управление пользователями
- Блокировка нарушителей с указанием причины
- Просмотр журнала системных событий

## Установка и запуск

### Требования
- Java 17+
- Maven 3.8+

### Запуск
```bash
git clone https://github.com/sss-creates/cargo-exchange
cd cargo-exchange
mvn clean javafx:run
```
